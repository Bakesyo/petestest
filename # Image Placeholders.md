# Image Placeholders

This directory should contain your website's images. For now, we're using online placeholder services (`https://placehold.co/`) in the HTML, but you should replace these with actual images.

## Recommended Image Files to Add:

1. **Logo Images:**
   - logo.png (200x80px) - Your company logo
   - favicon.png (32x32px) - Browser tab icon

2. **Hero/Banner Images:**
   - hero-banner.jpg (1200x600px) - Full-width banner image for the hero section

3. **Service Images:**
   - service-1.jpg (300x200px)
   - service-2.jpg (300x200px)
   - service-3.jpg (300x200px)

4. **About Images:**
   - about-us.jpg (500x400px)

5. **Portfolio/Project Images:**
   - project-1.jpg (350x250px)
   - project-2.jpg (350x250px)
   - project-3.jpg (350x250px)
   - project-4.jpg (350x250px)

## Image Optimization Tips

1. Compress all images using tools like [TinyPNG](https://tinypng.com/) or [ImageOptim](https://imageoptim.com/)
2. Use modern formats like WebP when possible
3. Ensure all images have appropriate alt text for accessibility
4. Consider using responsive images with the `srcset` attribute for different screen sizes
